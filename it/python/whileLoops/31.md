Cosa fa il codice `contatore += 1`?
```python
contatore = 1

while contatore < 5:
    print(contatore)
    contatore += 1
```

# --answers--

- Crea la variabile contatore
- Incrementa il contatore di due ogni volta
- Incrementa il contatore di tre ogni volta
