Una lista non deve avere una lunghezza fissa.
Puoi aggiungere elementi alla fine di una lista quando vuoi!
Per aggiungere un elemento ad una lista usiamo la parola chiave `append`:
```
>>> lettere = ["a", "b"]
>>> lettere.append("c")
>>> print(lettere)
['a', 'b', 'c']
```
