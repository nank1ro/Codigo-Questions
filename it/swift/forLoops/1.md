# --description--

Sappiamo come ripetere il codice usando un ciclo `while`.
Come questo programma che ripete le istruzioni per stampare `ciao`.
```swift
var contatore = 0

while contatore < 5 {
	print("ciao")
	contatore += 1
}
```
Ma possiamo fare lo stesso con i cicli `for`:
```swift
for i in 0..<5 {
	print("ciao")
}
```
