# --description--

Le variabili stringa possono essere dichiarate usando le doppie virgolette:
```swift
>>> let x = "Maggio"
```

# --instructions--

Dichiara una variabile denominata `mese` con il valore `giugno`

# --asserts--

`mese` deve essere uguale a `"giugno"`.

```swift
assert(mese == "giugno", "--err-t1--")
```
