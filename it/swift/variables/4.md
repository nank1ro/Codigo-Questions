# --description--

Quando aggiorniamo una variabile, essa dimentica il suo valore precedente.
Qui stampiamo due volte la variabile `x` e vediamo come cambia il suo valore
```swift
var x = 5
print(x) // prints 5
x = 10
print(x) // prints 10
```
