# --description--

Possiamo anche dare alle variabili i valori di altre variabili.
Qui, possiamo dare alla variabile `y` il valore di` x`
```swift
var x = 5
var y = x
print(y) // prints 5
```

# --instructions--

Crea una nuova variabile chiamata `y` e assegnale il valore della variabile` x`.
