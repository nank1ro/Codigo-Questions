# --instructions--

Usa `\()` per inserire il numero `500` nello spazio vuoto dell'interpolazione di stringa

# --seed--

```swift
print("Ci sono  oggetti")
```

# --solutions--

```swift
print("Ci sono \(500) oggetti")
```

# --output--

Ci sono 500 oggetti
