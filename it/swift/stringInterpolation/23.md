Cosa c'è di sbagliato nel codice?
```swift
let pezzi = 5
let label = "Abbiamo \(pezzi) pezzi"
```

# --answers--

- Non possiamo salvare le stringhe formattate nelle variabili
- I caratteri \() mancano
- Non possiamo usare le variabili all'interno di \()
- Nulla è sbagliato nel codice
