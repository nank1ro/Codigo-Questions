# --description--

In Swift possiamo far scorrere un array in un ciclo in un modo molto semplice usando le parole chiave `for..in`:
```swift
var numeri = [1, 2, 3]
for num in numeri {
	print(num)
}
// stampa 1, 2, 3 
```
Un nome di variabile segue la parola chiave `for`, ad essa verr√† assegnato il valore di ogni elemento dell'array a turno.
