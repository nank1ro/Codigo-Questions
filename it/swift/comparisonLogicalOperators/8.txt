Continuiamo con l'operatore logico **or** `||`.
L'operatore restituisce il primo operando che e' uguale a *true* o l'ultimo se tutti sono *false*.
```
print(2 == 2 || 2 == 3) // true
print(1 == 2 || 1 == 3) // false
```
