# --instructions--

Aggiungere il tipo appropriato per l'enumerazione in modo da poter iterare su di esso

# --seed--

```swift
enum Genere {
    case maschio, femmina
}
```

# --asserts--

`Genere` deve essere iterabile.

```swift
assert(Genere.allCases.count == 2, "--err-t1--")
```

# --solutions--

```swift
enum Genere: CaseIterable {
    case maschio, femmina
}
```
