# --description--

Ora vediamo gli operatori **logici**, iniziamo con il primo chiamato __AND__ `&&`.
L'operatore restituisce il primo operando che e' uguale a *false* o l'ultimo se tutti sono *true*.
```javascript
console.log(2 == 2 && 2 == 3);
// stampa false
console.log(1 == 1 && 1 == 1.0);
// stampa true
```
