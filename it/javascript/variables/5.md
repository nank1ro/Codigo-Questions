# --description--

In JavaScript le variabili stringa possono essere dichiarate sia usando le doppie virgolette che le singole:
```javascript
let x = "Maggio";
// entrambe sono la stessa stringa
let y = 'Maggio';
console.log(x === y);
// stampa true
```

# --instructions--

Dichiara una variabile denominata `mese` con il valore `"giugno"`
