<section id="description">
Un'array non deve avere una lunghezza fissa.
Puoi aggiungere elementi alla fine di un'array quando vuoi!
Per aggiungere un elemento ad un'array usiamo la funzione `push()`:
```
var lettere = ["a", "b"];
lettere.push("c");
console.log(lettere);
// stampa ["a", "b", "c"]
```
