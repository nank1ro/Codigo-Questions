# --description--

In JavaScript possiamo far scorrere un array in un ciclo in un modo molto semplice usando le parole chiave `for..of`:
```javascript
var numeri = [1, 2, 3];
for (num of numeri) {
    console.log(num);
}
// prints 1, 2, 3 
```
