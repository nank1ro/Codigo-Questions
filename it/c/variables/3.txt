<section id="description">
Possiamo anche dare alle variabili i valori di altre variabili. Qui, possiamo dare alla variabile `y` il valore di `x`
```
>>> int x = 5;
>>> int y = x;
>>> printf("%i\n", y);
5
```
</section>
