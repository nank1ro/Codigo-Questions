# --description--

Le stringhe formattate ci consentono di mostrare espressioni come stringhe con numeri, senza ricevere alcun errore.
Per mostrare un numero intero usiamo l'espressione `%i`

# --seed--

```c
printf("[/] amici\n", [/]);
```

# --answers--

- %i
- ?i
- %0
- 5

# --solutions--

```c
printf("%i amici\n", 5);
```

# --output--

5 amici
