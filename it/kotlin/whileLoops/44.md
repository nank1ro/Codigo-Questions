# --instructions--

Stampa i numeri __pari__ tra _1_ e _10_ (inclusi) utilizzando il ciclo _while_.

> SUGGERIMENTO: Potresti usare la parola chiave `continue` per saltare i numeri __dispari__.

# --solutions--

```kotlin
var numero = 0

while (numero < 11) {
    numero += 1
    if (numero % 2 != 0) continue
    println(numero)
}
```
