# --description--

Continuiamo con l'operatore di comparazione **non uguale** `!=`.
Restituisce un valore **booleano** (`true` o `false`) affermando se due espressioni **NON** sono uguali, ad esempio:
```kotlin
println(2 != 2) // false
println(2 != 3) // true
```
E' esattamente l'opposto dell'operatore *uguale*
