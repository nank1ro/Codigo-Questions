# --instructions--

Indica il modo corretto di convertire la variabile `numero` in un intero.
```dart
final numero = 3.14;
```

# --answers--

- ```dart
final intero = numero.toInt();
```
- ```dart
final intero = int(numero);
```
- ```dart
final int intero = numero;
```
- ```dart
final intero = numero as int;
```
