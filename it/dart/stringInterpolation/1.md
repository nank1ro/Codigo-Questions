# --description--

L'_interpolazione_ di stringhe è un modo programmatico per generare una stringa.
In Kotlin possiamo usare il segno `+` (concatenazione) per mostrare 2 o più stringhe assieme, ad esempio:
```dart
print("Hello " + "Dart!");
// stampa "Hello Dart!"
```
