# --instructions--

Usa `${}` per inserire il numero `500` nello spazio vuoto usando l'interpolazione di stringhe.

# --seed--

```dart
void main() {
    print("Ci sono  oggetti");
}
```

# --solutions--

```dart
void main() {
    print("Ci sono ${500} oggetti");
}
```

# --output--

Ci sono 500 oggetti
