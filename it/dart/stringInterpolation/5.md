# --description--

Se ciò che segue il segno `$` non è riconoscibile come un identificatore di programma, riscontrerai un errore

# --instructions--

Esegui per vedere l'errore

# --seed--

```dart
void main() {
    print("$5 amici");
}
```

# --solutions--

```dart
void main() {
    print("$5 amici");
}
```
