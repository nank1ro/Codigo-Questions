# --description--

In Kotlin we can use the `for-in` also for iterable collections calling the given closure on each element:
```kotlin
// this is a list, we'll see about that soon
val numbers = listOf(2, 4, 6, 8, 10) 
for (num in numbers) {
	println(num)
}
// prints (2, 4, 6, 8, 10)
```
