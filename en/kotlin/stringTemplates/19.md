# --instructions--

Use `${}` to insert the number `500` in the empty space of the string template

# --seed--

```kotlin
fun main() {
    println("There are  items")
}
```

# --solutions--

```kotlin
fun main() {
    println("There are ${500} items")
}
```

# --output--

There are 500 items
