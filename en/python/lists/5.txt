<section id="description">
A list doesn't have to have a fixed length.
You can add items to the end of a list any time you like!
To add an item to a list we use the `append` keyword:
```
>>> letters = ["a", "b"]
>>> letters.append("c")
>>> print(letters)
['a', 'b', 'c']
```
