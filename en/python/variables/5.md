# --description--

String variables can be declared either by using single or double quotes:
```python
>>> x = "May"
>>> x = 'May'
```
Both are the same thing.

# --instructions--

Declare a variable named `month` with the value of `june`.

# --asserts--

`month` must be equal to `"june"`.

```python
assert month == "june", "--err-t1--"
```

# --solutions--

```python
month = "june"
```
