# --instructions--

How do we know the `while` loop skips its code block?
```c
bool is_loading = false;

while (is_loading) {
    printf("Loading...\n");
}
```

# --answers--

- The variable in the condition is compared to true
- The variable in the condition is compared to false

# --solutions--
