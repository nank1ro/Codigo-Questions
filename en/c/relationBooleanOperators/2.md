# --description--

Let's continue with the **not equal** `!=` relational operator.
It returns a **boolean**, true `1`  or false `0`, stating whether two expressions are **NOT** equal, for example:
```c
>>> 2 != 2
0
>>> 2 != 3
1
```
It is exactly the opposite of the *equal* operator
