Now let's see the **boolean** operators, let's start with the first called __and__ `&&`.
It returns the first operand that evaluates to *false* or the last one if all are *true*.
```
>>> 2 == 2 && 2 == 3
0
>>> 1 == 1 && 1 == 1.0
1
```
