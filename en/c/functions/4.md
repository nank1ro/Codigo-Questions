# --description--

Functions can have multiple input parameters, which are written within the function's parentheses, separated by commas.
```c
void say_hello(char *name, bool new_user) {
  char greet[40] = "Hello ";
  strcat(greet, name);
  if (new_user) {
    strcat(greet, "! Welcome on board :)");
  }
  printf("%s\n", greet);
}

int main(void) {
    // prints "Hello Tom"
    say_hello("Tom", true);
    return 0;
};
```
