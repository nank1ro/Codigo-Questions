<section id="description">
When we update a variable, it forgets its previous value.
Here we can display the `x` variable twice and see how its value updates.
```
>>> int x = 5;
>>> printf("%i\n", x);
5
>>> x = 10;
>>> printf("%i\n", x);
10
```
</section>
