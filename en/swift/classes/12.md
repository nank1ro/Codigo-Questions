# --instructions--

Can you make `Dog` a subclass of `Animal`?
`legs` should be inside the `Animal` class while `breed` should be inside the `Dog` class

# --answers--

- class Animal {
-     var legs = 4
- }
- class Dog: Animal {
-     var breed: String?
- }
- let max = Dog()
- max.breed = "Chihuahua"
- print(max.breed!)
