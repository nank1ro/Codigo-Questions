# --instructions--

Add the appropriate type for the enumeration in order to be able to iterate over it

# --seed--

```swift
enum Gender {
    case male, female
}
```

# --asserts--

`Gender` must be iterable.

```swift
assert(Gender.allCases.count == 2, "--err-t1--")
```

# --solutions--

```swift
enum Gender: CaseIterable {
    case male, female
}
```
