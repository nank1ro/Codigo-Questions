<section id="description">
An array doesn't have to have a fixed length.
You can add items to the end of an array any time you like!
To add an item to an array we use the `append` function:
```
var letters = ["a", "b"]
letters.append("c")
print(letters)
// Prints ["a", "b", "c"]
```
