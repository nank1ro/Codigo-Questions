# --description--

In Swift we can loop through an array in a very simple way using the `for..in` keywords:
```swift
var numbers = [1, 2, 3]
for num in numbers {
	print(num)
}
// prints 1, 2, 3 
```
A variable name follows the `for` keyword, it will be assigned the value of each array item in turn.
