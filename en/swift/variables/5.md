# --description--

In Swift string variables can be declared only by using double quotes:
```swift
let x = "May"
```

# --instructions--

Declare a variable named `month` with the value of `june`.

# --asserts--

`month` must be equal to `"june"`.

```swift
assert(month == "june", "--err-t1--")
```

# --solutions--

```swift
let month = "june"
```
