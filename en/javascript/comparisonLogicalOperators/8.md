# --description--

Let's continue with the **or** `||` logical operator.
It returns the first operand that evaluates to *true* or the last one if all are *false*.
```javascript
console.log(2 == 2 || 2 == 3);
// prints true
console.log(1 == 2 || 1 == 3);
// prints false
```
