# --description--

Let's continue with the **less than or equal to** `<=` comparison operator.
It returns a **boolean** (`true` or `false`) stating whether one expression is less than or equal to the other, for example:
```javascript
console.log(2 <= 2); 
// prints true
console.log(3 <= 2);
// prints false
console.log(3 <= 4);
// prints true
```
