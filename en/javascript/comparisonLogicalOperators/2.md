# --description--

Let's continue with the **not equal** `!=` comparison operator.
It returns a **boolean** (`true` or `false`) stating whether two expressions are **NOT** equal, for example:
```javascript
console.log(2 != 2);
// prints false
console.log(2 != 3); 
// prints true
```
It is exactly the opposite of the *equal* operator
