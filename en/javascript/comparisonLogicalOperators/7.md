# --description--

Now let's see the **logical** operators, let's start with the first called __AND__ `&&`.
It returns the first operand that evaluates to *false* or the last one if all are *true*.
```javascript
console.log(2 == 2 && 2 == 3);
// prints false
console.log(1 == 1 && 1 == 1.0);
// prints true
```
