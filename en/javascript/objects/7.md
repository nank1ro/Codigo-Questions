# --description--

As for arrays, we can loop between dictionary elements using the `Object.entries()` method.
```javascript
var user = {"first_name": "John", "last_name": "Hood", "age": 30};
for (const [key, value] of Object.entries(user)) {
    console.log(`${key}: ${value}`);
}
```
