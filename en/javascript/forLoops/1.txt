<section id="description">
We know how to repeat code using a `while` loop.
Like this program repeating statements to display `hello`
```
var counter = 0;

while (counter < 5) {
	console.log("hello");
	counter++;
}
```
But we can do the same with `for` loops:
```
for (let i = 0; i < 5; i++) {
	console.log("hello");
}
```
