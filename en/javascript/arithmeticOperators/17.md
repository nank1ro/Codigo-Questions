# --instructions--

You have `60` products in the `store`, during the day you sold `50` of them and `bought` `55` new.
Update the `store` quantity.

# --seed--

```javascript
var store = 60;
var sold = 50;
var bought = 55;
// write after this line
```

# --asserts--

`store` must be equal to `65`.

```javascript
console.assert(store == 65, "--err-t1--");
```

# --solutions--

```javascript
store += bought - sold;
```
