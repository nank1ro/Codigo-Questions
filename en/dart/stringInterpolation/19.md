# --instructions--

Use `${}` to insert the number `500` in the empty space using string interpolation 

# --seed--

```dart
void main() {
    print("There are  items");
}
```

# --solutions--

```dart
void main() {
    print("There are ${500} items");
}
```

# --output--

There are 500 items
