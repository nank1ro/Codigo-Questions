# --instructions--

Indicate the correct way to convert the `number` variable to an integer.
```dart
final number = 3.14;
```

# --answers--

```dart
final integer = number.toInt();
```

```dart
final integer = int(number);
```

```dart
final int integer = number;
```

```dart
final integer = number as int;
```
